# AI-Powered Search & Rescue Robot for Bangladesh

## Overview
This project proposes an AI-powered autonomous search and rescue robot designed for disaster management in Bangladesh.

## Features
- YOLOv8 AI Vision
- Thermal Camera
- LiDAR + SLAM
- CO₂ Gas Sensor
- LoRa Communication
- Autonomous Navigation
- Disaster Mapping

## Research Basis
The design is developed by analyzing five published research papers and adapting their best features for Bangladesh.
